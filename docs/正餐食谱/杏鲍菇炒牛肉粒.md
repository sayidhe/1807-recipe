# 杏鲍菇炒牛肉粒

```
#小炒 #冬天
```

## 主料

- 杏鲍菇
- 牛肉粒或妈妈肉
- 菜椒，红黄个半颗
- 四季豆（可选）

## 配料

- 蒜粒
- 酱油
- 料酒

## 准备步骤

1. 杏鲍菇滚刀切；
2. 牛肉粒用酱油、料酒事先腌制好；
3. 菜椒切小方块；

## 烹饪步骤 

1. 热油，加蒜粒、牛肉粒炒一会儿，1分钟（若有四季豆可加入）；
2. 倒入杏鲍菇，和牛肉粒炒至均匀，然后加些许酱油和料酒，炒一会儿后加入些许水，盖上锅盖；
3. 闷5分钟左右，这时候也差不多出汁了；
4. 倒入菜椒，和锅里的牛肉粒、杏鲍菇搅拌均匀；
5. 在闷5分钟左右，加入调料，可以起锅啦。
