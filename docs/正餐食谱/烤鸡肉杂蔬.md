# 烤鸡肉杂蔬

```
#冬天 #烤箱 #圣诞音乐🎄 #麻烦程度3⭐️ #好吃程度5⭐️
```

## 主料

- 鸡腿肉
- 洋葱、番茄、蘑菇、四季豆
- 可适量加入土豆

## 配料

- 蚝油、生抽、孜然粉（可选）、橄榄油
- 花椒、蒜头、黑胡椒

## 准备步骤

1. 播放圣诞音乐；
2. 鸡腿肉过清水洗净，沥干；
3. 加入蚝油、生抽、橄榄油拌匀后；
4. 加入洋葱、花椒、黑胡椒、蒜头继续搅拌一会儿；
5. 预热烤箱5分钟；

## 烹饪步骤

1. 烤盘中放入锡箔纸；
2. 在底层倒入些许橄榄油；
3. 依次加入鸡腿肉、番茄、蘑菇、四季豆；
4. 锡箔纸包紧后放入烤箱；
5. 加热二十分钟左右，间隔五分钟，再次加热十分钟，即可。
