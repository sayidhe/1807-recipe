# 白葡萄酒柠檬白蛤

```
#一人食 #麻烦程度2⭐️ #好吃程度5⭐️
```

## 主料

- 白蛤蜊

## 配料

- 白葡萄酒
- 柠檬、葱花（大蒜苗）、大蒜、辣椒
- 黄油、橄榄油、盐、鸡精

## 准备步骤

1. 白蛤蜊洗净；
2. 将葱切小段（2cm）；
3. 大蒜切碎；
4. 辣椒切段；

## 烹饪步骤

1. 热锅中倒入橄榄油；
2. 大蒜切碎，倒入油中稍作翻炒；
3. 将白蛤蜊倒入锅中，稍作翻炒；
4. 倒入白葡萄酒，稍微淹没白蛤蜊；
5. 加入小块黄油，辣椒；
6. 煮3-5分钟后，挤入柠檬汁；
7. 最后，加葱段，放调料，即可出锅
