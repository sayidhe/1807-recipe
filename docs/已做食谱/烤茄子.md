# 烤茄子

```
#烤箱 #麻烦程度4⭐️ #好吃程度4⭐️
```

## 用具

- 烤箱、锡纸、油刷

## 主料

- 茄子两根

## 配料

- 蒜、姜、辣椒、葱
- 油、酱油、醋、糖、盐、味精

## 步骤

1. 烤箱230度预热 15 分钟；
2. 将茄子外层涂上油，并准备好锡纸包好；
3. 烤箱这时候差不多预热结束，将茄子放入，设置时间 20 分钟；
4. 此时可以用调料准备好酱汁；
5. 20分钟后取出茄子，用刀子对半切开，**并在中间用刀划几道口子**（更入味）；
6. 淋上酱汁，继续放入烤箱中，烤 10~15 分钟。
7. 完成。
