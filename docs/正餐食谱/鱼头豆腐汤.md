# 鱼头豆腐汤

```
#汤 #适合冬天 #滋补
```

## 主料

- 鲜鱼头
- 豆腐

## 配料

- 洋葱
- 大蒜
- 芹菜
- 金针菇（可选）

## 调料

- 盐
- 酒
- 白胡椒（可选）

## 准备步骤

1. 鱼头过开水3分钟；
2. 捞出后沥干水，准备烹饪

## 烹饪步骤 

1. 放油烧热后，将沥干水的鱼放入热油锅；
2. 在锅中炸一会儿之后，倒入料酒；
3. 可以放入些许蒜沫和洋葱沫增加香味和甜味；
4. 两三分钟后加入豆腐，继续大火烹饪；
5. 继续烹饪两三分钟，让豆腐和鱼的味道融合；
6. 加水，继续熬制20分钟左右；
7. 加入芹菜、金针菇、洋葱片；
8. 两三分钟后，加盐、味精等调料；
9. 起锅，完毕。
