# 土豆沙拉 

*WithEating 食帖 15 期 便当灵感集*

```
#二人食 #夏天 #麻烦程度3⭐️ #好吃程度5⭐️
```

## 食材

- 黑虎虾（去皮） 10-20只

## 调料

- 盐 1/2茶匙
- 黑胡椒粉 适量
- 大蒜 1颗
- 香草（欧芹、罗勒） 适量
- 橄榄油 2 汤匙

## 准备步骤

1. 虾肉上撒少许盐，静置15分钟，将渗出的水分用厨房纸巾吸干；
2. 密封容器中加入切碎的蒜片、香草、橄榄油与虾肉，混合均匀，放入冰箱冷藏一夜；

## 烹饪步骤

1. 平底锅加热，倒入虾肉，撒少许盐和黑胡椒粉，快速翻炒至虾被热透，颜色通红即可。

## Tips

- 其他贝壳类海鲜也可以同理来制作，只是需要提前焯水去壳
