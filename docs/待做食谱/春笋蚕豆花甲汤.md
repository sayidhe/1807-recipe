# 春笋蚕豆花甲汤

```
#麻烦程度 5⭐️ #好吃程度 5⭐️ #鲜味十足
```

## 食材

- 排骨
- 春笋（建瓯春笋）
- 蚕豆
- 花甲

## 配料

- 配料 1
- 配料 2

## 准备步骤

1. 竹笋剥皮后，滚刀切；
2. 花甲加点盐和冰之后泡水，吐沙；
3. 蚕豆剥好皮；
4. 排骨焯水

## 烹饪步骤

1. 砂锅烧热油，雷笋和蚕豆炒出香味；
2. 加入开水；
3. 加入焯过水的排骨；
4. 小火煮半小时；
5. 花甲放入，继续煮 10 分钟；
6. 放入些许胡椒粉和葱花；
7. 完成
