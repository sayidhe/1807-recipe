# 煎牛排

```
#西餐 #麻烦程度2⭐️ #好吃程度5⭐️
```

## 材料

- 牛排 Prime

## 配料

- 橄榄油、黄油
- 百里香、蒜末
- 海盐

## 准备步骤

1. 将新鲜牛排解冻，洗净后，用海盐和黑胡椒涂抹，腌制10分钟；
2. 把百里香洗净，准备5个大蒜

## 烹饪步骤

1. 加入些许猪油，将油烧热（足够热）；
2. 放入腌制好的牛排，双面煎到金黄；
3. 加入黄油，等黄油融化后不断把油泼在牛排上；
4. 放入百里香和大蒜，盖上盖子煎一分钟左右，让牛排内部熟一些；
5. 将牛夹入盘子中后，往锅中加入一小勺水（酱料用）；
6. 继续烧酱料，待锅中的酱料金黄和黏浊之后，淋在盘中的牛排上；
7. 切块，摆盘。
